# Work packets

This directory is the workflow-of-record for planned work.

Canonical packet path:
- `docs/work/<id>-<slug>/`

Status is represented by a small pointer file stored in:
- `docs/work/_queue/<status>/<id>-<slug>.md`

See `docs/WORKFLOW.md` and `docs/adr/ADR-0005-work-packets-and-queue.md`.
